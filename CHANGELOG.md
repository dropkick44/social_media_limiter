## [0.1.0] - 2026-02-21

### Added

- Initial release
- Menu bar application with status indicators
- Daily time budget with shared pool across all blocked sites
- System-level blocking via `/etc/hosts`
- Browser detection for Safari, Chrome, Firefox, and Arc
- Override mechanism with configurable delay and grant time
- Sleep/shutdown aware reset logic
- Add/remove blocked sites from menu
- Basic settings dialog for daily limit
